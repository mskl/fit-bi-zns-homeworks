IF nejde točit pedály AND řetěz neleží na ozubených kolečkách THEN nahoď spadlý řetěz
IF kolo prázdné AND plášť není roztržený THEN píchnutá duše
IF píchnutá duše AND díra není velká THEN použij lepení
IF píchnutá duše AND díra v duši je velká nebo uniká ventilek THEN vyměň duši
IF kolo prázdné AND plášť je roztržený THEN vyměň plášť i duši
IF zvuk od přehazovačky AND při šlapání THEN uprav natažení lanka stavěcím šroubem na řidítkách
IF řetěz přeskakuje THEN uprav stavěcí šroub přehazovačky
IF řetěz přeskakuje AND řetěz starý AND správně seřízený přesmykač THEN vyměň řetěz
IF řetěz padá v krajních polohách THEN uprav dorazy stavěcími šrouby
IF řetěz přetržený AND řetěz nebyl poslední dobou vyměněn THEN snýtuj řetěz
IF řetěz přetržený AND řetěz ve špatném stavu THEN vyměň řetěz za nový a snýtuj
IF zvuk při brždění AND staré brzdové špalky THEN vyměň brzdové špalky
IF zvuk při brždění AND špinavé brzdové špalky THEN umyj brzdové špalky
IF zvuk při jízdě AND od kola THEN jedno z kol dělá zvuk
IF jedno z kol dělá zvuk AND v pravidelných intervalech THEN vycentruj kolo
IF jedno z kol dělá zvuk AND konstantně THEN kolo není správně zasazeno ve vidlici/rámu
IF zvuk při jízdě AND od řetězu AND při šlapání AND skřípání THEN promaž řetěz a vodící prvky
IF brzda nefunguje AND brzda je volná THEN vyměň přetržené brzdové lanko
IF brzda nefunguje AND brzda klade odpor AND staré brzdové špalky THEN vyměněn brzdové špalky
IF nefunguje řazení AND řadící lanko je volné THEN vyměň přetržené řadící lanko
